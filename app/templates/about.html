About

What is containR?

containR was developed in 2018 at Harvard University by Christopher Chen (as an undergraduate thesis project) to improve the reproducibility of research code written in R. To achieve this goal, this website creates Docker images containing all the code, data, and package dependencies for R-based research projects, ensuring that the R code contained within will run on any computer with Docker installed. For usage instructions, check out the instructions page.
